How to use Api

In this demo, i will be using python request library to send http request to the server and convert html document to pdf. here is the program.

              
import requests

token = "90cb162608ae115e10bd4751786b70b131d2ff9b"

data = {'token':token}
files = {'file': open('index.html','rb')}

r = requests.post("http://localhost:8000/api-convert/",files = files,data = data)

if r.status_code == 200:
  with open("index.pdf","wb") as file:
    file.write(r.content)
